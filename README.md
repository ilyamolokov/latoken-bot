## Шаги для запуска бота

Установите Node.js версии `v18.20.4`

Установите зависимости:

```bash
npm install
# или
yarn install
```

В корне проекта создайте файл `.env` и добавьте в него следующие переменные:

```bash
OPENAI_API_KEY=
TELEGRAM_BOT_TOKEN=
```

Запустите бота:

```bash
npm run dev
# или
yarn dev
```
